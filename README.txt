# Ahmed Hadramaut EasyOrder System Presentation

Open `index.html` in a browser, or put the folder in VS Code and use Live Server.

Controls:
- Right Arrow / Space: Next slide
- Left Arrow: Previous slide
- Home button: First slide
- N: Toggle speaker notes
- F: Fullscreen

To replace diagrams later, keep the same file names inside the `assets` folder:
- activity-place-order.jpg
- activity-receive-order.jpg
- activity-track-orders.jpg
- activity-update-menu.jpg
- activity-generate-report.jpg
- sequence-place-order.jpg
- sequence-receive-order.jpg
- sequence-track-orders.jpg
- sequence-generate-report.jpg

Color palette followed:
#10232A, #3D4D55, #A79E9C, #D3C3B9, #B58863, #16161E

This version was updated so that each use case has one complete explanation slide.
